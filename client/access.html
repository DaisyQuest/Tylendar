Permission playbook

Use these permission tiers to keep collaborations secure while still giving teammates the access they need.

  • View Calendar — See full event details and attendee lists.
  • Times Only — View time blocks without event descriptions.
  • Add to Calendar — Create events and manage events they created.
  • Comment on Calendar — Leave feedback and join conversations.
  • Manage Calendar — Update or delete events and adjust access.

Governance checklist

Keep your team aligned with policy expectations.

  • Confirm every manager has a backup owner.
  • Audit recurring shares for external vendors monthly.
  • Review time-only access before publishing public availability.
  • Validate comments are enabled only for internal collaborators.

Audit readiness

Make sure changes to permissions are traceable and approved.

  • Track permission edits in the audit feed.
  • Require approval for new share links.
  • Export permission history for compliance reviews.
